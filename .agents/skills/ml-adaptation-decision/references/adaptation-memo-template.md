# Adaptation Memo Template

## Core Questions

1. What is the task and failure cost?
2. What is the current baseline behavior?
3. Is the main problem knowledge, behavior, or task fit?
4. Which lightest method could plausibly work?
5. What cost, latency, and maintenance burden does each option add?
6. What evaluation should follow the decision?
7. When should the team escalate to a heavier method?

## Suggested Output

```md
Task:

Main Failure Cost:

Recommended Adaptation Path:

Why Simpler Options Are or Are Not Enough:

Grounding / Knowledge Needs:

Latency / Cost Tradeoff:

Operational Risks:

Evaluation Plan:

Exit Criteria for Escalation:
```
