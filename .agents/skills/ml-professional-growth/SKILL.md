# ML Professional Growth

Use this skill when career development should become a visible engineering system instead of a mood-driven aspiration.

## What This Skill Does

This skill produces:

- current strengths
- current weaknesses
- one specialization direction
- one portfolio artifact to create
- one communication habit to improve
- one review checkpoint

## Workflow

1. Review what you can already explain and build.
2. Review where your projects still feel shallow.
3. Choose one growth direction, not many.
4. Design one artifact that would visibly prove improvement.
5. Reassess after shipping it.

## Output Format

Use this structure:

- `Current Strengths`
- `Current Weaknesses`
- `Recommended Specialization Direction`
- `Portfolio Artifact to Create`
- `Communication Habit to Improve`
- `Next Review Checkpoint`
- `Why This Direction`

## Quality Bar

- Prefer evidence from artifacts over self-description alone.
- Choose one growth direction, not five.
- Suggest an artifact that makes growth visible to other people.
- Include one communication habit because engineering growth is not only technical.

## Good Prompt Shapes

- Review my current ML profile and tell me what to build next.
- Turn these project artifacts into a growth memo.
- What specialization direction should I explore next based on my recent work?
