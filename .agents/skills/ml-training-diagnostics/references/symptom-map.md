# Symptom Map

## Oscillating or Diverging Loss

Likely causes:

- learning rate too high
- unstable gradients
- bad scaling

## Training Improves, Validation Collapses

Likely causes:

- overfitting
- split mismatch
- leakage or label problems

## Nothing Learns

Likely causes:

- broken data path
- wrong loss/task setup
- learning rate too low
- bad initialization
