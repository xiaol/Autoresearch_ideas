# Architecture Note Template

## Core Questions

1. What problem pressure is this architecture responding to?
2. What did the old method struggle with?
3. What mechanism changed?
4. What gain is claimed?
5. What cost or tradeoff appears?
6. Where should the change help?
7. Where might it not matter?
