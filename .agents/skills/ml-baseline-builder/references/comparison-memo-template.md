# Baseline Comparison Memo Template

## Core Questions

1. What is the exact task and target?
2. What is the simplest naive baseline?
3. What is the first interpretable model?
4. Which metric should compare them?
5. Which slices matter most?
6. What evidence would justify a richer model?
7. What evidence says the richer model is not yet justified?
