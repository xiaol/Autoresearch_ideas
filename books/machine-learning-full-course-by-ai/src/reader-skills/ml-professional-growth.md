# Reader Skill: ML Professional Growth

This page mirrors the real local skill used in the repository.

Use it after [Chapter 16](../chapter-16/README.md) whenever growth should be driven by evidence, artifacts, and review rather than vague ambition.

Real skill source:

`.agents/skills/ml-professional-growth/SKILL.md`

Invoke in Codex:

`$ml-professional-growth`

## What This Skill Does

This skill produces:

- current strengths
- current weaknesses
- one specialization direction
- one portfolio artifact to create
- one communication habit to improve
- one review checkpoint

## Workflow

1. Review what you can already explain and build.
2. Review where your projects still feel shallow.
3. Choose one growth direction, not many.
4. Design one artifact that would visibly prove improvement.
5. Reassess after shipping it.

## Output Format

- `Current Strengths`
- `Current Weaknesses`
- `Recommended Specialization Direction`
- `Portfolio Artifact to Create`
- `Communication Habit to Improve`
- `Next Review Checkpoint`
- `Why This Direction`

## Try It Now

```text
Use $ml-professional-growth to review my current ML profile. My recent artifacts are a fraud model project, a retrieval demo, and a launch-readiness memo. Tell me my current strengths, current weaknesses, one specialization direction to explore next, one portfolio artifact to create, one communication habit to improve, and the next review checkpoint.
```
