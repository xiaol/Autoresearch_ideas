# Knowledge Search Retrieval Brief

We want a very small company-knowledge search example that readers can run locally.

## Goal

Show how retrieval evaluation works with:

- a tiny document set
- a tiny query set
- a simple bag-of-words retrieval method
- explicit top-1 accuracy
- failure inspection

## Why this case matters

This is not a production retriever. It is a runnable teaching example that makes retrieval evaluation concrete before readers scale up to embeddings and larger indexes.
