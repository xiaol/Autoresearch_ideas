# Runnable Example Cases

This folder contains small local examples that readers can execute directly.

## Cases

### 1. Delivery-Time Prediction

Path:

`delivery-time-prediction/`

Run:

```bash
python3 examples/delivery-time-prediction/scripts/run_baseline.py
```

What it demonstrates:

- a tiny tabular dataset
- a naive baseline
- a simple linear regression baseline
- the kind of artifacts used by `ml-baseline-builder` and `ml-tabular-model-review`

### 2. Knowledge Search Retrieval

Path:

`knowledge-search-retrieval/`

Run:

```bash
python3 examples/knowledge-search-retrieval/scripts/run_retrieval_eval.py
```

What it demonstrates:

- a tiny retrieval corpus
- a tiny query set
- a simple lexical retrieval scorer
- the kind of artifacts used by `ml-retrieval-evaluation`
