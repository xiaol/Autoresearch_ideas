# Lesson Outline

- Lesson title: First Models: Linear Models and Nearest Neighbors
- Lesson goal: establish baselines as serious engineering tools
- Anchor case: delivery-time prediction
- Main skill or harness: ML Baseline Builder
- Primary visual: naive baseline -> interpretable baseline -> richer model decision
- On-screen artifact: delivery example run output
- Viewer takeaway: complexity should earn its place

## Teaching Flow

1. Open with the temptation to jump straight to a complex model.
2. Show why naive and interpretable baselines matter.
3. Run the delivery-time example.
4. Compare the naive baseline to the linear baseline.
5. End with the Baseline Builder skill and the one-page comparison memo.
