# Chapter 4 Pilot Lesson Pack

This chapter is the first gold-standard lesson package for the course.

Use it as the template for how later chapters should be prepared.

## Goal

Turn Chapter 4 into a lesson that is ready for:

- screen recording
- voice narration
- slide or scene assembly
- later YouTube release

## What This Lesson Demonstrates

- why baselines are serious engineering tools
- how to run one real local example
- how to interpret a simple baseline result
- how to close with a reusable skill, not only a number

## Core Assets

- `lesson-outline.md`
- `voiceover.md`
- `scene-cards.md`
- `demo-commands.md`
- `shot-list.md`
- `recording-checklist.md`
- `assets/title-card.svg`
- `assets/process-figure.svg`
- `assets/evidence-panel.svg`

## Recording Flow

1. Show the title card for the lesson opening.
2. Use `assets/process-figure.svg` during the concept explanation.
3. Run the delivery example from `demo-commands.md`.
4. Show `assets/evidence-panel.svg` while discussing the result.
5. Close with the Baseline Builder prompt and the engineering takeaway.

## Recommended Duration

- 8 to 10 minutes for a focused lesson
- 10 to 12 minutes if you want a slower teaching pace

## Standard For Future Chapters

A chapter should be considered video-ready when it has:

- one runnable or inspectable case
- one clear process figure
- one evidence visual
- one final voiceover draft
