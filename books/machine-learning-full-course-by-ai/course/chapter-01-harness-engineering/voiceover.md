# Voiceover Draft

## Opening

Machine learning is no longer learned in a quiet environment with only notes, code, and a search engine. We now study inside active systems with AI assistants, retrieval, critique loops, and reusable workflows.

## Core Shift

That changes the job of the learner. It is no longer enough to ask smart questions. We have to build a structure in which good work is easier to repeat and shallow work is harder to confuse with progress.

## Key Vocabulary

A prompt is a single request. A workflow is a repeated sequence. A skill packages that sequence so it can be reused. A harness is the broader system that makes the skill dependable in real work.

## Why It Matters

This is the main thesis of the course. The future professional is not only someone who can train models. It is someone who can build reliable human-AI working systems around models.

## Close

As you go through this course, do not only collect explanations. Build your own library of skills, review habits, and evidence trails.
