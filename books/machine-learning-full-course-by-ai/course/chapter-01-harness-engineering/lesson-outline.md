# Lesson Outline

- Lesson title: Learn Machine Learning Through Harness Engineering
- Lesson goal: reset the learner's mental model from isolated study to reusable human-AI systems
- Anchor case: a learner using AI constantly but still making unreliable progress
- Main skill or harness: the prompt -> workflow -> skill -> harness ladder
- Primary visual: the harness ladder and personal ML operating system
- On-screen artifact: Appendix B skill catalog plus one live skill invocation
- Viewer takeaway: the future ML engineer builds reliable working systems around learning and modeling

## Teaching Flow

1. Open with the problem of fast but unreliable AI-assisted learning.
2. Define prompts, workflows, skills, and harnesses.
3. Show why a reusable skill is different from a one-off chat.
4. Connect the idea to the learner-to-engineer growth ladder.
5. End by asking the viewer to identify one workflow they should package into a skill.
