# Lesson Outline

- Lesson title: Trees, Ensembles, and Strong Baselines
- Lesson goal: teach honest model-family comparison for tabular data
- Anchor case: fraud detection and structured risk scoring
- Main skill or harness: ML Tabular Model Review
- Primary visual: linear vs tree vs ensemble comparison table
- On-screen artifact: benchmark-style comparison memo
- Viewer takeaway: tabular problems deserve strong baseline families before prestige-driven escalation

## Teaching Flow

1. Open with why trees and ensembles still dominate many tabular tasks.
2. Explain splits, nonlinearity, and ensemble gains.
3. Compare model families honestly, not by reputation.
4. Show slice behavior and leakage concerns.
5. End with the Tabular Model Review skill.
