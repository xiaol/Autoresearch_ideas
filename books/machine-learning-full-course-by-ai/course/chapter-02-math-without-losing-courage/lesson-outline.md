# Lesson Outline

- Lesson title: Math Without Losing Courage
- Lesson goal: turn intimidating notation into operational understanding
- Anchor case: delivery-time prediction and the gradient update behind a simple model
- Main skill or harness: ML Math Translator
- Primary visual: formula-to-meaning-to-code flow
- On-screen artifact: one formula, one tiny code mapping, one worked example
- Viewer takeaway: math becomes usable when every symbol is tied to shape, operation, and code

## Teaching Flow

1. Open with why ML notation scares beginners.
2. Reframe vectors, matrices, gradients, and loss as operational language.
3. Translate one real update rule step by step.
4. Connect the formula to a small code view.
5. End by showing how the Math Translator skill forces reusable understanding.
